# birdspotter-web
Bird Detection Web App using AI

## Project Overview
This project is a Bird Detection system developed as a diploma final year project.
It uses Artificial Intelligence to detect birds using image and audio inputs.

## Frontend
- Built using Lovable AI
- Provides image upload and audio upload interface
- Displays bird detection results

## Backend
- Backend logic and AI models are maintained using GitHub
- API-based architecture is used to connect frontend and backend

## Technologies Used
- Lovable AI (Frontend)
- Python
- Flask
- Artificial Intelligence / Machine Learning
- GitHub (Version Control)
